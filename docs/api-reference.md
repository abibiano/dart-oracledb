# Public API Reference — oracledb (dart-oracledb)

> Generated by the Document Project workflow (deep scan), 2026-06-12. The public
> surface is exactly what `lib/oracledb.dart` re-exports — everything under
> `lib/src/` that is *not* re-exported is private implementation detail.

```dart
import 'package:oracledb/oracledb.dart';
```

## Public surface (from `lib/oracledb.dart`)

| Type / symbol | Source | Purpose |
|---------------|--------|---------|
| `OracleConnection` | `connection.dart` | The connection and its operations |

## `OracleConnection`

Created only via the static factories (constructor is private; a
`@visibleForTesting` `forTesting` constructor exists for tests).

### Opening

```dart
static Future<OracleConnection> connect(
  String connectString, {        // 'host:port/service', e.g. 'localhost:1521/FREEPDB1'
  required String user,
  required String password,
  TlsConfig? tlsConfig,
  int statementCacheSize = 30,
  bool preserveTimestampTimeZone = false,
  Duration? timeout,
});

static Future<T> withConnection<T>(
  String connectString, {
  required String user,
  required String password,
  required Future<T> Function(OracleConnection conn) callback,
  // ...same options as connect...
});  // opens, runs callback, always closes (try/finally)
```

> Exact optional-parameter names/defaults are defined at
> [connection.dart:884](../lib/src/connection.dart#L884) (`connect`) and
> [connection.dart:985](../lib/src/connection.dart#L985) (`withConnection`).

### Executing

```dart
Future<OracleResult> execute(String sql, [Object? bindValues]);
```

- `bindValues` is a `Map<String, Object?>` for **named** binds (`:name`) or a
  `List<Object?>` for **positional** binds (`:1`, `:2`, …).
- IN binds are plain Dart values; OUT / IN OUT binds are `OracleBind` instances.
- **One in-flight `execute` per connection** — an overlapping call is rejected
  (use a separate connection for concurrency).

### Transactions

```dart
Future<T> runTransaction<T>(Future<T> Function(OracleConnection conn) action);
Future<void> commit({Duration timeout = const Duration(seconds: 30)});
Future<void> rollback({Duration timeout = const Duration(seconds: 30)});
```

`runTransaction` runs `action`, commits on success, rolls back on throw. Nested
`runTransaction` is **not** supported (throws).

### Lifecycle

```dart
Future<bool> ping({Duration timeout = const Duration(seconds: 5)});
Future<void> close();
```

### Test-only instrumentation (`@visibleForTesting`, not public API)

`statementCacheSize`, `debugCacheSize`, `debugPendingCloseCount`,
`debugFullParseExecutes`, `debugReuseExecutes`, `debugLobReadOps`,
`debugPendingTempLobCount`, and the TTC sequence getter — used by integration
tests to assert protocol behaviour without privileged Oracle views. Production
code must not depend on them.

## `OracleResult` / `OracleRow`

```dart
result.rows;             // List<OracleRow>  (SELECT)
result.rowCount;         // int
result.columns;          // List<ColumnMetadata>
result.columnNames;      // List<String>
result.rowsAffected;     // int? — DML count; null for SELECT, and null (not 0) on pre-12.2 servers
result.outBinds;         // OracleOutBinds — PL/SQL OUT/IN OUT values
result.moreRowsAvailable;// bool — true if the FETCH cap or a missing cursor truncated the rows

row['NAME'];  row['name'];  row[0];   // by column name (case-insensitive) or index; null if absent/NULL
row.toList(); row.toMap();
```

**Oracle NULL convention:** an empty `VARCHAR2` (`''`) round-trips as `null`, not
a zero-length string (`'' IS NULL` in Oracle).

## Binds — `OracleBind`, `OracleDbType`, `OracleOutBinds`

```dart
OracleBind.out({ required OracleDbType type, int? maxSize });
OracleBind.inOut({ required Object? value, required OracleDbType type, int? maxSize });
```

`maxSize` is **required** for `varchar`, `raw`, `clob`, `blob`, `json` (and must
be `> 0`); numeric/date/timestamp use fixed protocol sizes. IN OUT values are
type-checked at construction (fails at the call site, not deep in wire encoding).

**`OracleDbType` and its `maxSize` units / Dart mapping:**

| `OracleDbType` | Dart type | `maxSize` unit | Notes |
|----------------|-----------|----------------|-------|
| `number` | `int` / `double` | — | NUMBER can exceed Dart `int`; rejects NaN/Inf |
| `varchar` | `String` | bytes | requires `maxSize` |
| `date` | `DateTime` | — | 7-byte |
| `timestamp` | `DateTime` | — | 11-byte, µs precision |
| `timestampTz` | `DateTime` (UTC) or `OracleTimestampTz` | — | offset preserved only with `preserveTimestampTimeZone: true` |
| `raw` | `Uint8List` | bytes | requires `maxSize` |
| `clob` | `String` | **characters** (UTF-16 units) | wire = LOB locator; `'' → NULL` |
| `blob` | `Uint8List` | bytes | wire = LOB locator; empty `Uint8List` → empty BLOB (≠ NULL) |
| `json` | `Map<String,Object?>` / `List<Object?>` | **OSON bytes** | Oracle 21c+; members: null/bool/finite num/String/nested map+list |

**Reading OUT binds:**

```dart
result.outBinds['name'];   // by name (case-insensitive); null if absent or SQL NULL
result.outBinds[0];        // by zero-based index over OUT/IN OUT outputs only
result.outBinds.toList();  result.outBinds.toMap();
// keys must be int or String, else ArgumentError
```

## `OracleException`

```dart
final int     errorCode;   // numeric ORA code (e.g. 1017)
final String  message;     // human-readable; ≤200-char snippet for query errors
final Object? cause;       // preserved originating error
final String? sql;         // full SQL (placeholders kept; bind values never substituted) — query errors only
final int?    offset;      // char offset into sql reported by Oracle — query errors only
String get code;           // 'ORA-01017' (5-digit pad floor); negative → 'ORA-invalid(<code>)', never throws
```

**Well-known code constants** (selection): `oraInvalidCredentials` (1017),
`oraConnectionClosed` (3113), `oraNetworkError` (12150), `oraConnectTimeout`
(12170), `oraConnectionRefused` (12514), `oraHostUnreachable` (12541),
`oraTlsHandshakeFailed` (28860), `oraTlsCertificateError` (28862),
`oraBindMismatch` (1008), `oraBindTypeError` (6502); plus protocol codes
`oraMalformedPacket`, `oraProtocolViolation`, `oraUnsupportedType`,
`oraDataTypeNotSupported`.

## `OracleTimestampTz`

Wrapper that preserves the original UTC offset of a `TIMESTAMP WITH TIME ZONE`
(default decode collapses to a UTC `DateTime`). Opt in per connection with
`preserveTimestampTimeZone: true`; it then applies to both column decode and
OUT/IN OUT binds. See [oracle_timestamp_tz.dart](../lib/src/oracle_timestamp_tz.dart).

## `TlsConfig`

```dart
TlsConfig({ bool enabled, bool verifyCertificate = true, SecurityContext? securityContext });
```

Pass to `connect`/`withConnection` to negotiate TLS before the TNS handshake.
`verifyCertificate: false` accepts self-signed certs (development only).

## Minimal example

```dart
import 'package:oracledb/oracledb.dart';

await OracleConnection.withConnection(
  'localhost:1521/FREEPDB1',
  user: 'system',
  password: 'oracle',
  callback: (conn) async {
    final r = await conn.execute(
      'SELECT id, name FROM employees WHERE dept = :d',
      {'d': 10},
    );
    for (final row in r.rows) {
      print('${row['ID']}: ${row['NAME']}');
    }

    await conn.runTransaction((tx) async {
      await tx.execute(
        'INSERT INTO employees (id, name, dept) VALUES (:i, :n, :d)',
        {'i': 1, 'n': 'Ada', 'd': 10},
      );
    });
  },
);
```

See `example/example.dart` and the README "Usage" section for PL/SQL, LOB, JSON,
and TLS examples.
